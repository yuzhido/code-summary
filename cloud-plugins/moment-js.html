格式化日期 ​

js
moment().format('MMMM Do YYYY, h:mm:ss a') // 二月 17日 2025, 3:20:44 下午
moment().format('dddd') // 星期一
moment().format('MMM Do YY') // 2月 17日 25
moment().format('YYYY [escaped] YYYY') // 2025 escaped 2025
moment().format() // 2025-02-17T15:20:44+08:00

设置 moment.js 一周从周一开始 ​

moment.js 默认一周从周日开始，我们如果想让时间默认从周一开始只用设置一下时区就可以了。

js
// require 方式
require('moment/locale/zh-cn')
moment.locale('zh-cn', { week: { dow: 1 } })

js
